1. Start BalenaEtcher

  1. Navigate to the folder where the neccesary files are located.

  2. Open/run balenaetcher portable by double clicking on it, or right clicking and selecting “Open”.

  3. Windows “Security Warning” pop-up will display on screen. Click “Run” to allow balenaetcher portable to run.

  4. Insert the USB flash drive into the computer.

  5. Balenaethcer portable should successfull open and present the screen below after accouple seconds.

Â  Â 

2. Select File to Flash

  1. Click on the “Flash from File” on the left of the menu to select the ISO to flash onto the USB flash drive.

Â  Â 

3. Select Target Drive

  1. Confirm the prefered USB flash drive is the selected option. Click on “change” in the center of the menu to view the drives available to flash menu.

Â  Â 

4. Flash Drive

  1. To flash the USB flash drive with the Linux ISO image, Click on “Flash!” on the right of the menu.

  2. Windows UAC “User Account Control” pop-up will display on screen. Enter your credentials, and click “Yes” to allow balenaetcher portable to begin flashing.

  • Wait for the program to verify the ISO image, flash the file onto the USB flash drive, and then verify the files that where flashed onto usb flash drive.
  • The time of completion depends on the ISO file size and the USB transfer speeds.
  • When the message above displays the flashing is done, and you can now confidently go to using the USB flash drive as you intend.
